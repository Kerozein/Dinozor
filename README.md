# Dinozor

## Description

Adaptation 3D du T. Rex Game

### Technologies
![npm](https://img.shields.io/npm/v/vite?color=orange&label=Vite&style=for-the-badge)

## Installation

````shell
git clone https://github.com/Kerozein/Dinozor.git
cd Dinozor
npm i
````
### Développement

````shell
npm run dev
````

### Déploiement

````shell
npm run build
````
